  Regional Eastern/Central/Mountain/Pacific P1 and P2
      Assigned by ma.TimeZone ('EST','CST','MST','PST') and whether DsnTbl is in
      the Scheduler "P2 inclusion" list vs the earlier dedicated pipelines.
      P2-style tables include (per timezone, lists differ slightly): claims family,
      dose_excuse, uaresultdetail, GlobalPayor, PayerClient, feesched/FeeSched,
      CheckIn, EandM forms, Bills, payerclthistory, treatmentlevel, Orders, etc.
      Everything else for that timezone that is not Dart/Dose/Orders/Forms/INV/Notes/LAB
      and not in the P2 list → that timezone's **P1** parent.
      Exact lists: Scheduler/Program.cs lines ~38–57.

  SAMMSGlobal
      ELSE branch of the same CASE — mappings that do not match any rule above.

  Important: A table's **parent TaskName** can still be overridden or tasks skipped by
  later UPDATEs in Scheduler (RowState = 26) for LAB/PHC/V5 schema, etc.
